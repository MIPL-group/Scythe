Scythe

A Program-by-example System for SQL

View the Project on GitHub (Mestway/Scythe)

About

Scythe is a program-by-example system aiming at helping end-users to write SQL queries: users provide the system input-output examples (tables) and additional constants that may be used in the task to the system, and Scythe synthesizes a query that is consistent with the examples.

Current State of Scythe

Scythe is currently able to synthesize queries with rich features: join, filter, exists, group-by, aggregtion, left-join, subquery and more. We studied a wide range of Stack Overflow questions and Scythe can solve many problems from the forum. However, Scythe is still a research prototype tool, and we are still working hard on making it more practical: adding more features and allows eaier interaction with users.

Running instruction

Here are the steps to run the tool on an input-output example.

  1. Download the system jar file SimpleSynthesizer.jar.
  2. Prepare a file containing your input-output example and addition constants. The format of input file can be found here.
  3. Run the command below. In this command, "file" refers to the path of your example file, and the option "StagedEnumerator" referes to the state-of-art synthesis algorithm used in Scythe. 
    java -jar Scythe.jar file StagedEnumerator
Scythe will then print the synthesized query for you.
